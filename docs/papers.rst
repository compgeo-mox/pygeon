Papers Using PyGeoN
===================

This page lists academic papers and research that use PyGeoN.

Overview
--------

PyGeoN has been used in various research projects focusing on:

* Structure-preserving numerical methods
* Mixed-dimensional modeling
* Porous media simulation
* Geomechanics and flow coupling

Publications
------------

For a complete and up-to-date list of papers using PyGeoN, please visit:
